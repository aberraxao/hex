O Hex é um jogo de tabuleiro estratégico jogado com uma grade hexagonal. O objetivo é conectar dois lados opostos do
tabuleiro com uma cadeia contínua de peças coloridas. Cada jogador tem uma cor atribuída.

Intruções:
- O tabuleiro do Hex é composto por hexágonos.
- Dois jogadores participam do jogo, um vermelho e um azul.
- O jogador vermelho tem como objetivo conectar os lados esquerdo e direito do tabuleiro com uma cadeia de peças vermelhas,
enquanto o jogador azul tem como objetivo conectar os lados superior e inferior com uma cadeia de peças azuis.
- Os jogadores alternam a colocação de suas peças no tabuleiro. Cada jogador coloca uma peça de sua cor em uma célula vazia em cada turno.
- O jogo continua até que um dos jogadores tenha conectado com sucesso seus dois lados opostos com sua cadeia de peças.
Esse jogador é declarado vencedor
